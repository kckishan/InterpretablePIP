# Interpretable protein-protein interaction prediction based on sparse gated sequence encoder
This is the pytorch implementation of the InterpretablePIP.

Requirements 
* PyTorch
* python3.6
* sklearn
* scipy

### Example to run demo.
```
python example.py
```
